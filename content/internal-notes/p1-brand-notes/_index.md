## 4B. NO AI FINGERPRINTS

All content must read like a human wrote it. AI-generated text leaves detectable patterns that erode trust with technical audiences. Strip them on every pass.

| `\u200b` `\ufeff` | Zero-width space / BOM | Delete (invisible but detectable) |

### Banned Phrasing Patterns
These are AI tells. They signal generated content to anyone paying attention.

| Pattern | Example | Why It Fails |
| :--- | :--- | :--- |
| **"In today's [noun]"** | "In today's digital landscape" | Generic AI opener. Says nothing. |
| **"It's important to note"** | "It's important to note that..." | Filler. If it is important, just say it. |
| **"Leverage/utilize"** | "Leverage AI to utilize data" | Corporate bloat. Use "use." |
| **"Comprehensive/robust"** | "A comprehensive solution" | Meaningless modifiers. What does it actually do? |
| **"Seamless/seamlessly"** | "Seamlessly integrates" | Nothing is seamless. Describe what happens. |
| **"Cutting-edge/next-gen"** | "Cutting-edge AI technology" | Marketing noise. Name the actual technology. |
| **"Empower/elevate"** | "Empowering businesses to..." | Vague. State the result. |
| **"Dive deep/deep dive"** | "Let's dive deep into..." | AI transition phrase. Just start the section. |
| **"Game-changer"** | "This is a game-changer" | Overused to the point of meaninglessness. |
| **"At the end of the day"** | "At the end of the day, it..." | Filler phrase. Delete the whole sentence. |

### The Rule
If a sentence could appear in any company's marketing copy by swapping the product name, it is generic. Delete it. Write something only Digital Crunch would say.

## 6. THE DEFINITION OF DONE: THE HARDENED STRATEGIC FILTER

### I. The Blue Ocean Audit (Kim & Mauborgne)
* **The ERRC Check:** Does this asset explicitly **Eliminate** a common industry headache or **Create** a result that "Jr. Marketers" cannot replicate?
* **The "Anti-Agency" Filter:** If a standard marketing agency could have written this, it's a failure. Does this feel like **Engineering Infrastructure**?

### II. The SMILE & SCRATCH Audit (Watkins)
* **The SMILE Test:** Does the copy **Suggest** a result (Leads/Sales), use **Imagery** (24x7 systems), and trigger an **Emotional** relief (The "Adults are here" feeling)?
* **The "Tame" Killer:** Is the language bold? Replace passive words with active "Wolverine" verbs: **Shred, Harden, Automate, Secure.**
* **The "Curse of Knowledge" Check:** Keep the magic in the results; keep the complexity in the code.

### III. The Radical Differentiation Audit (Neumeier)
* **The "Only" Anchor:** Does this asset remind the prospect of the 25-year Cyber-Security DNA? 
* **The Staccato Hook:** Is the anchor-**AI Systems. Leads. Sales. 24x7.**-implied or present?
* **The Professional Wizard Persona:** Is the tone blunt and objective? If it sounds like a "sales pitch," delete it.

### IV. The Visual Narrative Audit (Duarte)
* **The "Stealth" Hierarchy:** Does the design lead with **Obsidian (#0E1219)**? 
* **The "100" Benchmark:** Does the page achieve a Google PageSpeed Score of **100**? This is the literal proof of engineering authority. 
* **Typography Contrast:** **Inter Tight** for power; **JetBrains Mono** for data and "Proof of Work."

## FONT REFERENCE (Actual Implementation)

### Site-Wide (Blowfish / Tailwind Defaults)
No custom web fonts are loaded globally. The site relies on the operating system's native font stack for fast rendering and zero FOIT/FOUT.

| Role | Font Stack |
| :--- | :--- |
| **Body / Headings** | `ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji"` |
| **Code / Pre** | `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace` |

On Windows this resolves to **Segoe UI** (body) and **Cascadia Mono / Consolas** (code). On macOS it resolves to **SF Pro** and **SF Mono**.

### Status Dashboard (Google Fonts - loaded only on that page)
| Role | Font |
