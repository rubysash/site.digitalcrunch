## Keyboard Shortcuts

### Navigation 
| Shortcut | Description |
|----------|-------------|
| `Ctrl + Alt + ?` | View all keyboard shortcuts |
| `Alt + 1-8` | Open app in shelf position 1-8 |
| `Alt + 9` | Open last app in shelf |
| `Alt + Tab` | Switch to next window |
| `Alt + Shift + Tab` | Switch to previous window |
| `Alt + [` | Dock window to left |
| `Alt + ]` | Dock window to right |
| `Alt + =` | Maximize window |
| `Alt + -` | Minimize window |

###  Browser
| Shortcut | Description |
|----------|-------------|
| `Ctrl + T` | New tab |
| `Ctrl + W` | Close current tab |
| `Ctrl + Shift + T` | Reopen last closed tab |
| `Ctrl + Tab` | Next tab |
| `Ctrl + Shift + Tab` | Previous tab |
| `Ctrl + 1-8` | Jump to tab 1-8 |
| `Ctrl + 9` | Jump to last tab |
| `Ctrl + N` | New window |
| `Ctrl + Shift + N` | New incognito window |
| `Ctrl + L` | Focus address bar |
| `Ctrl + R` | Reload page |
| `Ctrl + Shift + R` | Hard reload (clear cache) |
| `Alt + Left Arrow` | Back |
| `Alt + Right Arrow` | Forward |

### Text Editing
| Shortcut | Description |
|----------|-------------|
| `Ctrl + A` | Select all |
| `Ctrl + C` | Copy |
| `Ctrl + X` | Cut |
| `Ctrl + V` | Paste |
| `Ctrl + Shift + V` | Paste without formatting |
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + Backspace` | Delete previous word |
| `Alt + Backspace` | Delete next word |
| `Ctrl + Left/Right` | Move cursor by word |
| `Ctrl + Shift + Left/Right` | Select word |

### Screenshot
| Shortcut | Description |
|----------|-------------|
| `Ctrl + Show Windows` | Screenshot entire screen |
| `Ctrl + Shift + Show Windows` | Partial screenshot (select area) |
| `Ctrl + Alt + Show Windows` | Screenshot current window |

### System
| Shortcut | Description |
|----------|-------------|
| `Search + L` | Lock screen |
| `Ctrl + Shift + Q (twice)` | Sign out |
| `Ctrl + Shift + Refresh` | Rotate screen |
| `Ctrl + Shift + +` | Zoom in |
| `Ctrl + Shift + -` | Zoom out |
| `Ctrl + Shift + 0` | Reset zoom |
| `Alt + Search` | Toggle Caps Lock |
| `Ctrl + Alt + Z` | Enable ChromeVox (screen reader) |

### Files
| Shortcut | Description |
|----------|-------------|
| `Ctrl + S` | Save |
| `Ctrl + P` | Print |
| `Ctrl + O` | Open file |
| `Ctrl + F` | Find on page |
| `Ctrl + G` | Find next |
| `Ctrl + Shift + G` | Find previous |

### Launcher
| Shortcut | Description |
|----------|-------------|
| `Search` | Open launcher |
| `Search + type` | Search apps, files, web |
| `Ctrl + Alt + /` | Open Help |

**Note:** The `Search` key is the magnifying glass key (where Caps Lock is on standard keyboards). On some Chromebooks it's called the `Launcher` key. The `Show Windows` key looks like a rectangle with two lines (F5 position).
