A reference for Hugo static site development, deployment with Netlify/GitHub, DNS, performance, SEO, and related web technologies.

This glossary is organized into the following sections:

- [Accessibility](accessibility/) - Web accessibility including WCAG, ARIA, and inclusive design
- [Content Delivery Network](content-delivery-network/) - CDN, caching, and edge networks
- [Content & File Types](content-file-types/) - HTML, CSS, JavaScript, images, and data formats
- [Conversion Rate Optimization](conversion-rate-optimization/) - A/B testing, landing pages, and CRO metrics
- [Domain Name System](domain-name-system/) - DNS records, nameservers, and domain management
- [Hosting & Deployment](hosting-development/) - Netlify, CI/CD, and serverless functions
- [Hugo Architecture](hugo-architecture/) - Hugo site structure, layouts, and templates
- [Hugo & Build Tools](hugo-tools/) - Hugo, build tools, and preprocessors
- [SEO & Discovery](seo-aeo/) - Search optimization, structured data, and meta tags
- [SSL/TLS & Security](ssl-tls-security/) - HTTPS, certificates, and security headers
- [Static Site Concepts](static-site-concepts/) - JAMstack, SSGs, and headless CMS
- [User Input & Forms](user-input/) - Form handling, validation, and spam protection
- [Version Control](version-control/) - Git, GitHub, and version control workflows
- [Web Analytics](web-analytics/) - Analytics, tracking, and privacy-focused alternatives
- [Web Performance](web-performance/) - Core Web Vitals and optimization techniques
